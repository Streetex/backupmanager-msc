## 📝 Description 

A program for setting up automatic or manual backups of My Summer Car game files. The software includes customizable settings that allow, for example, sending notifications upon a completed backup, or specifying the name and location of the backups and the folders containing them. The program can also be used to load an earlier backup if something goes wrong with your current one, or if you lose some item.

---

## ⚡ Features

- Automatic file backup (You can set the time when the program will back up files).
- Manual file backup (Click to create the back up files).
- Customizable setting (Language, notifications, save directory, backup file name, folders name, auto-delete backups, auto-backup)

## 🚀 Installation & Usage 

### Prerequisites 
*   Windows 10 / Windows 11

### Setup
* Some antivirus programs block unknown programs from reading or writing files in certain folders. This program is completely safe, but antiviruses flag it as malicious because it modifies folders on your computer. You need to add an exception for the entire process or folder.
